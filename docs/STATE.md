## UX-решения, уже зафиксированные
- Тренировка = сессия, а не отдельное упражнение.
- Внутри тренировки должно быть несколько блоков/упражнений.
- Категории тренировок должны быть расширяемыми пользователем.
- Справочник упражнений должен быть привязан к категории.
- Бег и плавание тоже состоят из блоков/отрезков, а не только из общей длительности.
